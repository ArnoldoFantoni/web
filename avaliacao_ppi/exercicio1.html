O que é HTML

HTML é um documento de texto aonde informamos essas marcações com tags para esse texto e o servidor da web consegue ler esse texto.

Em outras palavras, o arquivo HTML é um arquivo de marcador de texto que enviamos para o serviço web conseguir entender o que deve apresentar e como apresentar para o usuário
