O que é um regra CSS ?

Uma regra CSS é um conjunto de instruções que define como os elementos HTML devem ser exibidos na tela.

Ela é composta por um seletor, uma propriedade e um valor.

Exemplo:

seletor {

propriedade: valor;

}

Nesse texto estamos utilizando a regra abaixo.

span{

color: blue;

font-size: 18px;

}

Isso é uma tag span O seletor p aplica as propriedades color: blue; e font-size: 18px; a todos os parágrafos da página.
