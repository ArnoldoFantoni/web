Formulários

O HTML possui uma tag form que possibilita submissão de dados informados dentro do formulário. a tag <form> possui além dos atributos globais e também atributo action="url", utilizado para definir o destino dos dados submetidos.

Exemplo: 
        <form action="">
            <input type="text" name="" id="">
            ...
        </form>

Table

A tag table é utilizada para identificação de tabela no HTML, independentemente em uma tabela podemos criar linhas (rows) utilizando a tag <tr> e celulas dentro das linhas utilizando a tag <td>

Exemplo: 
        <table>
            <tr>
                <td>Celula 1x1</td>
                <td>Celula 1x2</td>
            </tr>
            <tr>
                <td>Celula 2x1</td>
                <td>Celula 2x2</td>
            </tr>
        </table>
O código anterior define a tabela abaixo

Celula 1x1 Celula 1x2
Celula 2x1 Celula 2x2
